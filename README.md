# 行政人员管理系统-执法部门版

此系统是专门为行政机关执法部门开发的。

它包括一下几个内容：(1).执法单位信息录入；(2).奖惩记录；(3).培训记录； (4).调离记录

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
